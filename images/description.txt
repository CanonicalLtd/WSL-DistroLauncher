Ubuntu on Windows allows one to use Ubuntu Terminal and run Ubuntu
command line utilities including bash, ssh, git, apt and many more.

To use this feature, one first needs to use "Turn WIndows features on
or off" and select "Windows Subsystem for Linux", click OK, reboot,
and use this app.

The above step can also be performed using Administrator PowerShell
prompt:
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

After above is done simply use "ubuntu" command in cmd.exe or launch
Ubuntu from the start menu.
